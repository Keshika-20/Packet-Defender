Packet Defender is a Python-based firewall simulation system designed to implement key network security techniques such as IP whitelisting, blacklisting, protocol-based filtering, throttling, and time-based access control.

The system processes incoming requests through a multi-layered filtering mechanism, ensuring that only trusted and valid traffic is permitted while blocking unauthorized or potentially harmful access. This layered approach improves security, reduces misuse, and enhances overall traffic control.

A user-friendly graphical interface built using Tkinter enables both administrators and users to interact with the system efficiently. Administrators can manage firewall rules such as IP access and protocol permissions, while users can simulate network requests and observe real-time results.

In addition to the software implementation, the project is extended into Cisco Packet Tracer to simulate real-world firewall behavior using Access Control Lists (ACLs), demonstrating how these concepts are applied in practical networking environments.
